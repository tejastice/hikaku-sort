# HIKAKU_SORT（比較ソート）

二分探索アルゴリズムを使った効率的な順位付けツール

## 概要

HIKAKU_SORTは、ユーザーが要素を比較しながら順位付けを行うウェブアプリケーションです。二分探索つき挿入ソートアルゴリズムにより、最小限の比較回数で効率的にランキングを作成できます。

## 機能

- **直感的な比較UI**: 2つの要素（AとB）を横並びで表示し、どちらが上位かを選択
- **進捗管理**: 比較回数と進捗率をリアルタイムで表示
- **効率的なソート**: 二分探索アルゴリズムにより最小限の比較回数で完了
- **リアルタイム表示**: ランキングを比較ごとにリアルタイム更新

## 使い方

1. `index.html` をブラウザで開く
2. テキストエリアに比較したい項目を入力（1行1項目）
3. 「比較を開始」ボタンをクリック
4. 表示される2つの要素を比較してボタンをクリック
   - **Aの勝ち**: Aがより高順位
   - **Bの勝ち**: Bがより高順位
   - **引き分け**: 同順位
5. すべての比較が終わるとランキングが完成

## アルゴリズム

二分探索つき挿入ソートを採用。n個の要素をソートする場合、最悪ケースでΣ(log₂(i)) (i=2 to n) 回の比較で完了します。
